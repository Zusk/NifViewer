# Struct `MipMap`

Description of a mipmap within an NiPixelData object.

## Attributes
- **module**: `NiMain`
- **name**: `MipMap`
- **size**: `12`

## Fields
- **Width** (`uint`)
  - Width of the mipmap image.
- **Height** (`uint`)
  - Height of the mipmap image.
- **Offset** (`uint`)
  - Offset into the pixel data array where this mipmap starts.

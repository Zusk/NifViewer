# Enum `LightColor`

Used by NiLightColorControllers to select which type of color in the controlled object that will be animated.

## Attributes
- **name**: `LightColor`
- **storage**: `ushort`

## Values
- `option` `LC_DIFFUSE` (`value`=`0`) – Control the diffuse color.
- `option` `LC_AMBIENT` (`value`=`1`) – Control the ambient color.

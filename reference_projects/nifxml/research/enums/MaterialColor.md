# Enum `MaterialColor`

Used by NiMaterialColorControllers to select which type of color in the controlled object that will be animated.

## Attributes
- **name**: `MaterialColor`
- **storage**: `ushort`

## Values
- `option` `TC_AMBIENT` (`value`=`0`) – Control the ambient color.
- `option` `TC_DIFFUSE` (`value`=`1`) – Control the diffuse color.
- `option` `TC_SPECULAR` (`value`=`2`) – Control the specular color.
- `option` `TC_SELF_ILLUM` (`value`=`3`) – Control the self illumination color.

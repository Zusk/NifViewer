# Enum `AlphaFormat`

Describes how transparency is handled in an NiTexture.

## Attributes
- **name**: `AlphaFormat`
- **storage**: `uint`

## Values
- `option` `ALPHA_NONE` (`value`=`0`) – No alpha.
- `option` `ALPHA_BINARY` (`value`=`1`) – 1-bit alpha.
- `option` `ALPHA_SMOOTH` (`value`=`2`) – Interpolated 4- or 8-bit alpha.
- `option` `ALPHA_DEFAULT` (`value`=`3`) – Use default setting.

# Enum `CollisionMode`

The collision mode controls the type of collision operation that is to take place for NiCollisionData.

## Attributes
- **name**: `CollisionMode`
- **prefix**: `CM`
- **storage**: `uint`

## Values
- `option` `USE_OBB` (`value`=`0`) – Use Bounding Box
- `option` `USE_TRI` (`value`=`1`) – Use Triangles
- `option` `USE_ABV` (`value`=`2`) – Use Alternate Bounding Volumes
- `option` `NOTEST` (`value`=`3`) – Indicates that no collision test should be made.
- `option` `USE_NIBOUND` (`value`=`4`) – Use NiBound

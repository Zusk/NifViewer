# Enum `ColliderType`

This is used by the Floodgate kernel to determine which NiPSColliderHelpers functions to call.

## Attributes
- **name**: `ColliderType`
- **since**: `V20_5_0_0`
- **storage**: `uint`

## Values
- `option` `COLLIDER_PLANAR` (`value`=`0`)
- `option` `COLLIDER_SPHERICAL` (`value`=`1`)

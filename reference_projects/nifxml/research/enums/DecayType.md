# Enum `DecayType`

Describes the decay function of bomb forces.

## Attributes
- **name**: `DecayType`
- **storage**: `uint`

## Values
- `option` `DECAY_NONE` (`value`=`0`) – No decay.
- `option` `DECAY_LINEAR` (`value`=`1`) – Linear decay.
- `option` `DECAY_EXPONENTIAL` (`value`=`2`) – Exponential decay.

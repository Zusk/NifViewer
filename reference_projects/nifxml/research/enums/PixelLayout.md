# Enum `PixelLayout`

Describes the color depth in an NiTexture.

## Attributes
- **name**: `PixelLayout`
- **prefix**: `PX`
- **storage**: `uint`

## Values
- `option` `LAY_PALETTIZED_8` (`value`=`0`) – Texture is in 8-bit palettized format.
- `option` `LAY_HIGH_COLOR_16` (`value`=`1`) – Texture is in 16-bit high color format.
- `option` `LAY_TRUE_COLOR_32` (`value`=`2`) – Texture is in 32-bit true color format.
- `option` `LAY_COMPRESSED` (`value`=`3`) – Texture is compressed.
- `option` `LAY_BUMPMAP` (`value`=`4`) – Texture is a grayscale bump map.
- `option` `LAY_PALETTIZED_4` (`value`=`5`) – Texture is in 4-bit palettized format.
- `option` `LAY_DEFAULT` (`value`=`6`) – Use default setting.
- `option` `LAY_SINGLE_COLOR_8` (`value`=`7`)
- `option` `LAY_SINGLE_COLOR_16` (`value`=`8`)
- `option` `LAY_SINGLE_COLOR_32` (`value`=`9`)
- `option` `LAY_DOUBLE_COLOR_32` (`value`=`10`)
- `option` `LAY_DOUBLE_COLOR_64` (`value`=`11`)
- `option` `LAY_FLOAT_COLOR_32` (`value`=`12`)
- `option` `LAY_FLOAT_COLOR_64` (`value`=`13`)
- `option` `LAY_FLOAT_COLOR_128` (`value`=`14`)
- `option` `LAY_SINGLE_COLOR_4` (`value`=`15`)
- `option` `LAY_DEPTH_24_X8` (`value`=`16`)

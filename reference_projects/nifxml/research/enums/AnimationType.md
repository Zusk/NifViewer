# Enum `AnimationType`

Bethesda Animation. Animation type used on this position. This specifies the function of this position.

## Attributes
- **name**: `AnimationType`
- **storage**: `ushort`
- **versions**: `#BETHESDA#`

## Values
- `option` `Sit` (`value`=`1`) – Actor use sit animation.
- `option` `Sleep` (`value`=`2`) – Actor use sleep animation.
- `option` `Lean` (`value`=`4`) – Used for lean animations?

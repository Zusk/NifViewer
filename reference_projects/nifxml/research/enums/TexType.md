# Enum `TexType`

The type of texture.

## Attributes
- **name**: `TexType`
- **storage**: `uint`

## Values
- `option` `BASE_MAP` (`value`=`0`) – The basic texture used by most meshes.
- `option` `DARK_MAP` (`value`=`1`) – Used to darken the model with false lighting.
- `option` `DETAIL_MAP` (`value`=`2`) – Combined with base map for added detail.  Usually tiled over the mesh many times for close-up view.
- `option` `GLOSS_MAP` (`value`=`3`) – Allows the specularity (glossyness) of an object to differ across its surface.
- `option` `GLOW_MAP` (`value`=`4`) – Creates a glowing effect.  Basically an incandescence map.
- `option` `BUMP_MAP` (`value`=`5`) – Used to make the object appear to have more detail than it really does.
- `option` `NORMAL_MAP` (`value`=`6`) – Used to make the object appear to have more detail than it really does.
- `option` `PARALLAX_MAP` (`value`=`7`) – Parallax map.
- `option` `DECAL_0_MAP` (`value`=`8`) – For placing images on the object like stickers.
- `option` `DECAL_1_MAP` (`value`=`9`) – For placing images on the object like stickers.
- `option` `DECAL_2_MAP` (`value`=`10`) – For placing images on the object like stickers.
- `option` `DECAL_3_MAP` (`value`=`11`) – For placing images on the object like stickers.

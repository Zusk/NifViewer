# Enum `ApplyMode`

Describes how the vertex colors are blended with the filtered texture color.

## Attributes
- **name**: `ApplyMode`
- **storage**: `uint`

## Values
- `option` `APPLY_REPLACE` (`value`=`0`) – Replaces existing color
- `option` `APPLY_DECAL` (`value`=`1`) – For placing images on the object like stickers.
- `option` `APPLY_MODULATE` (`value`=`2`) – Modulates existing color. (Default)
- `option` `APPLY_HILIGHT` (`value`=`3`) – PS2 Only.  Function Unknown.
- `option` `APPLY_HILIGHT2` (`value`=`4`) – Parallax Flag in some Oblivion meshes.

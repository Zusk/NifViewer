# Enum `SortingMode`

Describes the way that NiSortAdjustNode modifies the sorting behavior for the subtree below it.

## Attributes
- **name**: `SortingMode`
- **storage**: `uint`

## Values
- `option` `SORTING_INHERIT` (`value`=`0`) – Inherit. Acts identical to NiNode.
- `option` `SORTING_OFF` (`value`=`1`) – Disables sort on all geometry under this node.

# Enum `BSCPCullingType`

Culling modes for multi bound nodes.

## Attributes
- **name**: `BSCPCullingType`
- **prefix**: `BSCP`
- **storage**: `uint`
- **versions**: `#FO3_AND_LATER#`

## Values
- `option` `CULL_NORMAL` (`value`=`0`) – Normal
- `option` `CULL_ALLPASS` (`value`=`1`) – All Pass
- `option` `CULL_ALLFAIL` (`value`=`2`) – All Fail
- `option` `CULL_IGNOREMULTIBOUNDS` (`value`=`3`) – Ignore Multi Bounds
- `option` `CULL_FORCEMULTIBOUNDSNOUPDATE` (`value`=`4`) – Force Multi Bounds No Update

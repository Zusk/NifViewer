# Enum `BSLightingShaderType`

Values for configuring the shader type in a BSLightingShaderProperty

## Attributes
- **name**: `BSLightingShaderType`
- **prefix**: `ST`
- **storage**: `uint`
- **versions**: `#SKY_AND_LATER#`

## Values
- `option` `Default` (`value`=`0`)
- `option` `Environment Map` (`value`=`1`) – Enables EnvMap Mask(TS6), EnvMap Scale
- `option` `Glow Shader` (`value`=`2`) – Enables Glow(TS3)
- `option` `Parallax` (`value`=`3`) – Enables Height(TS4)
- `option` `Face Tint` (`value`=`4`) – Enables Detail(TS4), Tint(TS7)
- `option` `Skin Tint` (`value`=`5`) – Enables Skin Tint Color
- `option` `Hair Tint` (`value`=`6`) – Enables Hair Tint Color
- `option` `Parallax Occ` (`value`=`7`) – Enables Height(TS4), Max Passes, Scale. Unimplemented.
- `option` `Multitexture Landscape` (`value`=`8`)
- `option` `LOD Landscape` (`value`=`9`)
- `option` `Snow` (`value`=`10`)
- `option` `MultiLayer Parallax` (`value`=`11`) – Enables EnvMap Mask(TS6), Layer(TS7), Parallax Layer Thickness, Parallax Refraction Scale, Parallax Inner Layer U Scale, Parallax Inner Layer V Scale, EnvMap Scale
- `option` `Tree Anim` (`value`=`12`)
- `option` `LOD Objects` (`value`=`13`)
- `option` `Sparkle Snow` (`value`=`14`) – Enables SparkleParams
- `option` `LOD Objects HD` (`value`=`15`)
- `option` `Eye Envmap` (`value`=`16`) – Enables EnvMap Mask(TS6), Eye EnvMap Scale
- `option` `Cloud` (`value`=`17`)
- `option` `LOD Landscape Noise` (`value`=`18`)
- `option` `Multitexture Landscape LOD Blend` (`value`=`19`)
- `option` `FO4 Dismemberment` (`value`=`20`)

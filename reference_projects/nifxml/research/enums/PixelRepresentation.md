# Enum `PixelRepresentation`

Describes how each pixel should be accessed on NiPixelFormat.

## Attributes
- **name**: `PixelRepresentation`
- **prefix**: `PX`
- **storage**: `uint`

## Values
- `option` `REP_NORM_INT` (`value`=`0`)
- `option` `REP_HALF` (`value`=`1`)
- `option` `REP_FLOAT` (`value`=`2`)
- `option` `REP_INDEX` (`value`=`3`)
- `option` `REP_COMPRESSED` (`value`=`4`)
- `option` `REP_UNKNOWN` (`value`=`5`)
- `option` `REP_INT` (`value`=`6`)

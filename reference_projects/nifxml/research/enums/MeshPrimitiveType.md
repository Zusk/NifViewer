# Enum `MeshPrimitiveType`

Describes the type of primitives stored in a mesh object.

## Attributes
- **name**: `MeshPrimitiveType`
- **since**: `V20_5_0_0`
- **storage**: `uint`

## Values
- `option` `MESH_PRIMITIVE_TRIANGLES` (`value`=`0`) – Triangle primitive type.
- `option` `MESH_PRIMITIVE_TRISTRIPS` (`value`=`1`) – Triangle strip primitive type.
- `option` `MESH_PRIMITIVE_LINES` (`value`=`2`) – Lines primitive type.
- `option` `MESH_PRIMITIVE_LINESTRIPS` (`value`=`3`) – Line strip primitive type.
- `option` `MESH_PRIMITIVE_QUADS` (`value`=`4`) – Quadrilateral primitive type.
- `option` `MESH_PRIMITIVE_POINTS` (`value`=`5`) – Point primitive type.

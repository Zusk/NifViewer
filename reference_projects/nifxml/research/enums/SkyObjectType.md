# Enum `SkyObjectType`

Sets what sky function this object fulfills in BSSkyShaderProperty or SkyShaderProperty.

## Attributes
- **name**: `SkyObjectType`
- **storage**: `uint`
- **versions**: `#BETHESDA#`

## Values
- `option` `BSSM_SKY_TEXTURE` (`value`=`0`) – BSSM_Sky_Texture
- `option` `BSSM_SKY_SUNGLARE` (`value`=`1`) – BSSM_Sky_Sunglare
- `option` `BSSM_SKY` (`value`=`2`) – BSSM_Sky
- `option` `BSSM_SKY_CLOUDS` (`value`=`3`) – BSSM_Sky_Clouds
- `option` `BSSM_SKY_STARS` (`value`=`5`) – BSSM_Sky_Stars
- `option` `BSSM_SKY_MOON_STARS_MASK` (`value`=`7`) – BSSM_Sky_Moon_Stars_Mask

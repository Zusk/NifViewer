# Enum `PixelTiling`

Describes whether pixels have been tiled from their standard row-major format to a format optimized for a particular platform.

## Attributes
- **name**: `PixelTiling`
- **prefix**: `PX`
- **storage**: `uint`

## Values
- `option` `TILE_NONE` (`value`=`0`)
- `option` `TILE_XENON` (`value`=`1`)
- `option` `TILE_WII` (`value`=`2`)
- `option` `TILE_NV_SWIZZLED` (`value`=`3`)

# Enum `CoordGenType`

Determines the way that UV texture coordinates are generated.

## Attributes
- **name**: `CoordGenType`
- **storage**: `uint`

## Values
- `option` `CG_WORLD_PARALLEL` (`value`=`0`) – Use planar mapping.
- `option` `CG_WORLD_PERSPECTIVE` (`value`=`1`) – Use perspective mapping.
- `option` `CG_SPHERE_MAP` (`value`=`2`) – Use spherical mapping.
- `option` `CG_SPECULAR_CUBE_MAP` (`value`=`3`) – Use specular cube mapping. For NiSourceCubeMap only.
- `option` `CG_DIFFUSE_CUBE_MAP` (`value`=`4`) – Use diffuse cube mapping. For NiSourceCubeMap only.

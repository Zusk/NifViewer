# Enum `LightingMode`

Describes which lighting equation components influence the final vertex color for NiVertexColorProperty.

## Attributes
- **name**: `LightingMode`
- **storage**: `uint`

## Values
- `option` `LIGHT_MODE_EMISSIVE` (`value`=`0`) – Emissive.
- `option` `LIGHT_MODE_EMI_AMB_DIF` (`value`=`1`) – Emissive + Ambient + Diffuse. (Default)

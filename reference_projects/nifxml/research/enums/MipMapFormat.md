# Enum `MipMapFormat`

Describes how mipmaps are handled in an NiTexture.

## Attributes
- **name**: `MipMapFormat`
- **storage**: `uint`

## Values
- `option` `MIP_FMT_NO` (`value`=`0`) – Texture does not use mip maps.
- `option` `MIP_FMT_YES` (`value`=`1`) – Texture uses mip maps.
- `option` `MIP_FMT_DEFAULT` (`value`=`2`) – Use default setting.

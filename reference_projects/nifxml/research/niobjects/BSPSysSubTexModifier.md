# Niobject `BSPSysSubTexModifier`

Similar to a Flip Controller, this handles particle texture animation on a single texture atlas

## Attributes
- **inherit**: `NiPSysModifier`
- **module**: `BSParticle`
- **name**: `BSPSysSubTexModifier`
- **versions**: `#BETHESDA#`

## Fields
- **Start Frame** (`float`)
  - Starting frame/position on atlas
- **Start Frame Fudge** (`float`)
  - Attributes: `default`=`64.0`
  - Random chance to start on a different frame?
- **End Frame** (`float`)
  - Attributes: `default`=`63.0`
  - Ending frame/position on atlas
- **Loop Start Frame** (`float`)
  - Frame to start looping
- **Loop Start Frame Fudge** (`float`)
- **Frame Count** (`float`)
  - Attributes: `default`=`30.0`
- **Frame Count Fudge** (`float`)

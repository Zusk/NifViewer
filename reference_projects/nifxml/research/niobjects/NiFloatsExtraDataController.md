# Niobject `NiFloatsExtraDataController`

Animates an NiFloatsExtraData object attached to an NiAVObject.
NiInterpController::GetCtlrID() string format:
    '%s[%d]'
Where %s = Value of "Extra Data Name", %d = Value of "Floats Extra Data Index"

## Attributes
- **inherit**: `NiExtraDataController`
- **module**: `NiAnimation`
- **name**: `NiFloatsExtraDataController`

## Fields
- **Floats Extra Data Index** (`int`)
- **Data** (`Ref`)
  - Attributes: `template`=`NiFloatData`, `until`=`10.1.0.103`

# Niobject `NiSortAdjustNode`

Used to turn sorting off for individual subtrees in a scene. Useful if objects must be drawn in a fixed order.

## Attributes
- **inherit**: `NiNode`
- **module**: `NiMain`
- **name**: `NiSortAdjustNode`

## Fields
- **Sorting Mode** (`SortingMode`)
  - Attributes: `default`=`SORTING_INHERIT`
  - Sorting
- **Accumulator** (`Ref`)
  - Attributes: `template`=`NiAccumulator`, `until`=`20.0.0.3`

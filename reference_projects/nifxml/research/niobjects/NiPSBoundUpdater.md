# Niobject `NiPSBoundUpdater`

Updates the bounding volume for an NiPSParticleSystem object.

## Attributes
- **inherit**: `NiObject`
- **module**: `NiPSParticle`
- **name**: `NiPSBoundUpdater`
- **since**: `V20_5_0_0`

## Fields
- **Update Skip** (`ushort`)
  - Number of particle bounds to skip updating every frame. Higher = more updates each frame.

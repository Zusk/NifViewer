# Bitflags `VertexAttribute`

The bits of BSVertexDesc that describe the enabled vertex attributes.

## Attributes
- **name**: `VertexAttribute`
- **prefix**: `VF`
- **storage**: `ushort`
- **versions**: `#SSE# #FO4# #F76#`

## Values
- `option` `Vertex` (`bit`=`0`)
- `option` `UVs` (`bit`=`1`)
- `option` `UVs_2` (`bit`=`2`)
- `option` `Normals` (`bit`=`3`)
- `option` `Tangents` (`bit`=`4`)
- `option` `Vertex_Colors` (`bit`=`5`)
- `option` `Skinned` (`bit`=`6`)
- `option` `Land_Data` (`bit`=`7`)
- `option` `Eye_Data` (`bit`=`8`)
- `option` `Instance` (`bit`=`9`)
- `option` `Full_Precision` (`bit`=`10`)

# Bitflags `BSPartFlag`

Editor flags for the Body Partitions.

## Attributes
- **module**: `BSMain`
- **name**: `BSPartFlag`
- **storage**: `ushort`
- **versions**: `#BETHESDA#`

## Values
- `option` `PF_EDITOR_VISIBLE` (`bit`=`0`) – Visible in Editor
- `option` `PF_START_NET_BONESET` (`bit`=`8`) – Start a new shared boneset.  It is expected this BoneSet and the following sets in the Skin Partition will have the same bones.
